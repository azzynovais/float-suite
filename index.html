∞ Why Float Suite

Zero Tracking

No analytics, no telemetry, no ads. Float Suite never touches a server. Everything runs entirely in your browser's memory and localStorage.

Offline First

Works without internet. All tools are standalone HTML files — bookmark them, save to disk, run them from a USB drive. No CDN failures, no 404s.

Pure Vanilla JS

No React, no bundlers, no build steps. Open DevTools and read the source. Every tool is a single self-contained HTML file you can fork and modify.

Your Data Stays Yours

Documents auto-save to localStorage. Export to SVG, PNG, PDF, HTML, or Markdown whenever you want. No vendor lock-in, ever.

Keyboard-First UX

Every tool has full keyboard shortcuts. Power users can do everything without touching the mouse — tool switching, undo/redo, nudging, selection, export.

Dark Mode Everywhere

All four apps support full dark mode with persistent preference. Your chosen theme is remembered across sessions via localStorage — zero flicker on load.
